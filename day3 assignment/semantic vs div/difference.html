difference between semantic vs div

Semantic tags clearly describe their meaning to both the browser and the developer.
Examples include <header>, <footer>, <article>, <section>, and <nav>.
These tags provide better accessibility, SEO, and code readability.

< div > is a generic container used to group elements for styling or scripting purposes.
It does not convey any specific meaning about its content.
While <div> is versatile, overusing it can lead to less readable code and poorer accessibility.
